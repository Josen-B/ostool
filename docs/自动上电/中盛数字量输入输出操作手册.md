# 数字量输入输出系列操作手册（RS485/RS232版）

## 1 产品接线

### 1.1 接线表

| 标识 | 功能 |
|------|------|
| + | 供电电源正极（6~36V/12V/24VDC）。8路及以下为宽电压6~36VDC；8路以上有12VDC和24VDC两种可选 |
| - | 供电电源负极 |
| A | RS485+ |
| B | RS485- |
| COM/CN | 输出公共端 |
| NO | 输出常开端 |
| NC | 输出常闭端 |
| XCOM | 输入信号公共端，接电源正极或电源负极。可直接从供电电源取电，也可独立接电源进行隔离；接电源正极（10~30VDC）为低电平触发（NPN）；接电源负极为高电平触发（PNP） |
| X | 信号输入端 |

### 1.2 数字量输入接线

**NPN型（低电平触发）：** XCOM 接电源正极（10~30VDC），信号输入端 X 通过开关接电源负极。

**PNP型（高电平触发）：** XCOM 接电源负极，信号输入端 X 通过开关接电源正极。

### 1.3 继电器输出接线

- COM/CN 为公共端
- NO 为常开端（正常状态下断开，触发时导通）
- NC 为常闭端（正常状态下导通，触发时断开）

### 1.4 晶体管输出接线

PNP型输出（控制电源正极），最大负载 3A，10~30VDC。

## 2 通讯协议

产品采用标准 **ModbusRTU** 协议，支持以下功能码：0x01、0x02、0x03、0x04、0x05、0x06、0x0F、0x10。

### 2.1 默认通讯参数

| 参数 | 默认值 |
|------|--------|
| 地址 | 1 |
| 波特率 | 38400 |
| 数据位 | 8 |
| 停止位 | 1 |
| 奇偶校验 | 无 |

### 2.2 寄存器类型

| 寄存器类型 | 用途 | 掉电保存 |
|-----------|------|---------|
| 输入寄存器 / 离散输入状态寄存器 | 保存输入口状态 | - |
| 保持寄存器（线圈） | 控制输出口开关 | 否 |
| 保持寄存器（参数） | 设置模块参数 | 是 |

## 3 寄存器功能定义

### 3.1 线圈寄存器（控制输出开关）

| 协议地址 | PLC地址 | 功能描述 |
|---------|---------|---------|
| 0000H | 00001 | 通道1控制：0-关闭，1-开启 |
| 0001H | 00002 | 通道2控制：0-关闭，1-开启 |
| 0002H | 00003 | 通道3控制：0-关闭，1-开启 |
| 0003H | 00004 | 通道4控制：0-关闭，1-开启 |
| ... | ... | 通道N控制 |
| 002FH | 00048 | 通道48控制：0-关闭，1-开启 |

### 3.2 离散输入状态寄存器（只读，输入状态）

| 协议地址 | PLC地址 | 功能描述 |
|---------|---------|---------|
| 0000H | 10001 | 通道1输入状态：0-未触发，1-已触发 |
| 0001H | 10002 | 通道2输入状态：0-未触发，1-已触发 |
| ... | ... | 通道N输入状态 |
| 002FH | 10048 | 通道48输入状态：0-未触发，1-已触发 |

### 3.3 输入寄存器（只读，输入状态）

| 协议地址 | PLC地址 | 功能描述 |
|---------|---------|---------|
| 0000H | 30001 | 通道1输入状态：0-未触发，1-已触发 |
| 0001H | 30002 | 通道2输入状态 |
| ... | ... | 通道N输入状态 |
| 002FH | 30048 | 通道48输入状态 |
| 0032H | 30051 | 按位表示通道1~16输入口状态（最低位=通道1） |
| 0033H | 30052 | 按位表示通道17~32输入口状态 |
| 0034H | 30053 | 按位表示通道33~48输入口状态 |

**读取示例（十六进制）：**

```
读通道1状态发送：01 04 00 00 00 01 31 CA
未触发时返回：  01 04 02 00 00 B9 30
已触发时返回：  01 04 02 00 01 78 F0
```

### 3.4 保持寄存器（输出控制与参数设置）

#### 3.4.1 通道控制寄存器

| 协议地址 | PLC地址 | 复位值 | 功能描述 |
|---------|---------|-------|---------|
| 0000H | 40001 | 0 | 通道1控制 |
| 0001H | 40002 | 0 | 通道2控制 |
| ... | ... | 0 | 通道N控制 |

每个通道具有 **5种工作模式**：

| 模式 | 说明 |
|------|------|
| **1. 普通模式** | 写0关闭，写1开启。写>1时：延时操作，延时时间 = (N-1) x 0.01秒 |
| **2. 联动模式** | 具有普通模式所有功能，且输入触发时自动开启同通道号输出，输入未触发时自动关闭 |
| **3. 点动模式** | 写1翻转输出状态（关->开，开->关） |
| **4. 开关循环模式** | 设置开关时间，时间 = (N-1) x 0.01秒 |
| **5. 开固定时长模式** | 设置开启时间后自动关闭，时间 = (N-1) x 0.01秒 |

#### 3.4.2 参数配置寄存器

| 协议地址 | PLC地址 | 复位值 | 功能描述 | 掉电保存 |
|---------|---------|-------|---------|---------|
| 0031H | 40050 | 0 | 输入口状态主动上传控制 | 是 |
| 0032H | 40051 | 1 | RS485总线地址/站号（1~255） | 是 |
| 0033H | 40052 | 4 | 波特率设置 | 是 |
| 0034H | 40053 | 0 | 批量控制：0-全关，1-全开 | - |
| 0035H | 40054 | 0 | 按位控制通道1~16（最低位=通道1） | - |
| 0096H | 40151 | 0 | 通道1控制模式设置 | 是 |
| 0097H | 40152 | 0 | 通道2控制模式设置 | 是 |
| ... | ... | 0 | 通道N控制模式设置 | 是 |
| 00C6H | 40198 | 0 | 通道48控制模式设置 | 是 |

**输入口状态主动上传控制（寄存器0031H）：**

| 值 | 功能 |
|----|------|
| 0 | 不主动上传（出厂默认） |
| 1 | 任一输入口状态变化时主动上传 |
| >1 | 按间隔主动上传，间隔 = (N-1) x 0.01秒 |

**波特率设置（寄存器0033H）：**

| 值 | 波特率 |
|----|-------|
| 0 | 4800 |
| 1 | 9600 |
| 2 | 14400 |
| 3 | 19200 |
| 4 | 38400（出厂默认） |
| 5 | 56000 |
| 6 | 57600 |
| 7 | 115200 |

**通道控制模式设置（寄存器0096H~00C6H）：**

| 值 | 模式 |
|----|------|
| 0 | 普通工作模式 |
| 1 | 联动模式（输入通道和指令均可控制） |
| 2 | 点动模式（翻转模式） |
| 3 | 开关循环模式 |
| 4 | 自动复位模式 |

> **注意：** 掉电保存参数修改后需重新上电生效。

## 4 指令示例

### 4.1 修改站号为2（06功能码）

```
发送：01 06 00 32 00 02 A9 C4
      |  |  |     |     |
      |  |  |     |     CRC校验码
      |  |  |     寄存器值=2
      |  |  寄存器地址=0032H（站号）
      |  功能码=06
      从站地址=01
```

### 4.2 修改波特率为9600（06功能码）

```
发送：01 06 00 33 00 01 B8 05
      寄存器地址=0033H（波特率），值=1（9600）
```

### 4.3 控制第一通道继电器吸合

**方式一：06功能码（写单个保持寄存器）**

```
发送：01 06 00 00 00 01 48 0A
      寄存器地址=0000H（通道1），值=1（开启）
```

**方式二：05功能码（写单个线圈）**

```
发送：01 05 00 00 FF 00 CD FD
      线圈地址=0000H（通道1），值=FF00（开启）
```

### 4.4 控制多个通道（10功能码写多个保持寄存器）

控制第一、第二通道同时吸合：

```
发送：01 10 00 00 00 02 04 00 01 00 01 63 AF
      |  |  |     |     |  |
      |  |  |     |     |  数据：通道1=1，通道2=1
      |  |  |     |     字节数=4
      |  |  |     寄存器个数=2
      |  |  起始地址=0000H
      |  功能码=10
      从站地址=01
```

### 4.5 控制第一至第四通道吸合（0F功能码写多个线圈）

```
发送：01 0F 00 00 00 04 01 0F 7E 92
      起始地址=0000H，线圈数=4，数据=0F（二进制00001111，通道1~4全开）
```

### 4.6 读取第一通道输入状态（04功能码）

```
发送：01 04 00 00 00 01 31 CA
      寄存器地址=0000H，读取个数=1

响应：01 04 02 00 01 78 F0
      数据=0001 表示已触发

响应：01 04 02 00 00 B9 30
      数据=0000 表示未触发
```

## 5 操作流程总结

### 5.1 基本使用步骤

1. **接线：** 按接线表连接电源、RS485通讯线、输入/输出信号线
2. **配置通讯参数：** 使用默认参数（地址1，波特率38400）或通过指令修改
3. **设置通道工作模式：** 写入对应控制模式寄存器（0096H~00C6H）
4. **控制输出：** 通过线圈寄存器或保持寄存器控制输出开关
5. **读取输入：** 通过输入寄存器或离散输入状态寄存器读取输入状态

### 5.2 常用操作速查

| 操作 | 功能码 | 寄存器地址 | 写入值 |
|------|--------|----------|--------|
| 开启通道1 | 06 | 0000H | 0001 |
| 关闭通道1 | 06 | 0000H | 0000 |
| 全部开启 | 06 | 0034H | 0001 |
| 全部关闭 | 06 | 0034H | 0000 |
| 读取通道1输入 | 04 | 0000H | 读取1个 |
| 修改站号 | 06 | 0032H | 1~255 |
| 修改波特率 | 06 | 0033H | 0~7 |
| 设置联动模式 | 06 | 0096H | 1 |
| 开启主动上传 | 06 | 0031H | 1 |
